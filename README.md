# KUMA HEADQUARTERS // SYS.COM

This repository hosts the source code for the **KUMA HEADQUARTERS** portal site. It serves as a central hub and system monitor linking to various contents and platforms.

## Features
- **Data Science**: Documentation and notes on data science, data analysis, and Python (Hosted on Netlify/MkDocs).
- **Data Apps (Main)**: Python-based data applications (Hosted on Streamlit).
- **Data Apps (Sub)**: External applications (Hosted on OCI).
- **Toukei Kentei Drill**: Statistics certification practice application (Hosted on Vercel).
- **GitHub Profile**: Portfolio, OSS contributions, and scripts.

The site also includes a lightweight system monitoring script that checks the online status of the linked platforms.
